# 🐾 Animal Guessing Game (Python)

## 📖 Description

This is a simple **console-based Animal Guessing Game** built using Python.
The computer randomly selects an animal, and the player must guess it based on a series of clues.

The game provides up to **5 clues**, and the player gets multiple chances to guess the correct animal. If the player fails after all clues, the correct answer is revealed.

---

## 🎮 How the Game Works

* A random animal is selected from a predefined list.
* The player is given clues one by one.
* After each clue, the player enters a guess.
* If the guess is correct → 🎉 You win!
* If all 5 clues are used and the answer is still incorrect → ❌ You lose.

---

## 🧠 Features

* Random animal selection using Python's `random` module
* Multiple animals with unique clues
* Interactive user input
* Step-by-step clue system

3. Open terminal or command prompt
4. Run the program:

